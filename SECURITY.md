# Security Policy

## Supported Versions

Security updates are provided for the latest released version of IntelliTask. Users are encouraged to upgrade to the most recent release to receive security fixes and dependency updates.

| Version | Supported |
| -------- | --------- |
| Latest release | ✅ |
| Older releases | ❌ |

## Reporting a Vulnerability

The security of IntelliTask and its users is taken seriously.

If you discover a security vulnerability, please do **not** create a public GitHub issue. Instead, report it privately by contacting the maintainer:

- **Maintainer:** Stefan-Mihai MOGA
- **Repository:** [IntelliTask](https://github.com/mihaimoga/IntelliTask)

When reporting a vulnerability, please include:

- A clear description of the issue
- Steps required to reproduce the vulnerability
- The affected IntelliTask version
- Any proof-of-concept code, screenshots, or logs
- Your assessment of the potential impact

## What to Expect

After a vulnerability report is received:

1. The report will be reviewed and validated.
2. The maintainer will assess its severity and impact.
3. A fix will be developed and tested when appropriate.
4. The reporter may be contacted for additional information.
5. Once remediation is available, a security update will be released.

## Responsible Disclosure

Please allow a reasonable amount of time for investigation and remediation before publicly disclosing any vulnerability. Coordinated disclosure helps protect IntelliTask users and the wider community.

## Scope

This policy applies to:

- IntelliTask source code
- Official IntelliTask releases
- Installation packages and portable distributions
- Third-party dependencies included with the project

Issues related to third-party libraries should also be reported to the respective upstream maintainers when appropriate.

## Security Best Practices for Users

To help keep your system secure:

- Download IntelliTask only from official project releases.
- Use the latest available version.
- Keep Windows and security software up to date.
- Verify downloaded files before installation when possible.
- Report any suspicious behavior immediately.

Thank you for helping make IntelliTask safer for everyone.
